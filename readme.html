Miniblog 0.16-COMPAT Readme

Install:

  • Extract the contents of miniblog.zip (or tar.gz if you downloaded it)
  • Place contents in your wp-content/plugins folder of your Wordpress Blog (the contents should be a folder called miniblog)
  • Go to your Wordpress admin area and activate miniblog
  • You all done! Go have fun!

Upgrade:

  • Go to your Wordpress admin area and deactivate any previous versions of Miniblog
  • Delete miniblog.php from your wp-content/plugins folder
  • Follow Install steps

If you have any problems please post them on the miniblog page. Thank you and Enjoy...
